= Code Blocks
:description: Examples of creating code blocks.

{description} More info available at https://docs.asciidoctor.org/asciidoc/latest/verbatim/source-blocks/.

We use Prism.js for source code highlighting. See all the available languages: https://prismjs.com/#supported-languages.
